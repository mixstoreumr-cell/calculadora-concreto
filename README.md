# Calculadora de Traço de Concreto CIMENZO

Calculadora profissional para cálculo de traço de concreto voltada para a produção de cubas, revestimentos 3D e peças decorativas.

## Funcionalidades

- ✅ Seleção de tipo de aplicação (Revestimento 3D, Cubas, Pisos, UHPC)
- ✅ Escolha entre Cimento CP5 e Cimento Branco
- ✅ Cálculo automático de insumos
- ✅ Pigmentação personalizada
- ✅ Gerenciamento de preços
- ✅ Edição de proporções base
- ✅ Cálculo de mão de obra
- ✅ Exportação para PDF e Excel
- ✅ Funciona offline (PWA)
- ✅ Responsivo para celular, tablet e desktop

## Como Usar

1. Acesse a calculadora
2. Selecione o tipo de aplicação
3. Escolha o tipo de cimento
4. Insira o peso do cimento em kg
5. Defina a pigmentação (%)
6. Adicione o preço de mão de obra
7. Visualize os resultados automaticamente

## Abas Disponíveis

## Instalação como App

### Android

2. Toque no menu (⋮)
3. Selecione "Instalar app"

### iOS/iPadOS
1. Acesse o site
2. Toque em "Compartilhar"
3. Selecione "Adicionar à Tela Inicial"
